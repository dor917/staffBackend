## 1.프로젝트 소개

Stp-Project는 GitHub의 어려운 복잡한 이론과 세팅을 해야 한다는 단점을 보완을 하기위해 만든 웹사이트 입니다. 
Github의 번거로운 세팅 과정과 안에 복잡한 로직을 미니 프로젝트를 하는 
초보개발자들에게 맞게 간소화 하였으며 팀원들 간의 협동을 중심으로 일정을 조율하거나 프로젝트 코드의 변경사항을 알 수 있는 기능도 추가하였습니다.
